Getting Started with HTML

Glossary for Udacity: Intro to Programming NanoDegree

HTML

HyperText Markup Language is the heart of the web. It is made up text content, markup,refrerences, and links to other documents.

Tag

Tags are the building blocks of markup. A tag is an Element, which consist of 3 parts. An opening tag, content, and closing tag.

Attributes

An attribute assigns a value to a tag via a "name". Important to note that tags can have multiple attributes.

Whitespace

Whitespace in HTML renders as a single space.

Inline Element

Inline Elements are pure text. They render elements in the same line. They can indicate a new line or a line break, but they do not have the capacity for values like height or width. Even img tags are inline elements. Span is inline

Block Element

Block Elements create an invisible box, which can have attached values like height or width. Div is block.

DOCTYPE

indicates the type of html.

Head

The head contains references/links for meta-data, javascript, and css information for the html document.

Body

The body contains the contents of an HTML document.
